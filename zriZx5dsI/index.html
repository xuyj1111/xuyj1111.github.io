【Java】final关键字

2022-04-28 1 min read # Java # 原创

可以用来修饰：类 、变量、方法，不能用来修饰抽象类和接口

final变量

final变量只能被赋值一次，赋值后值不再改变

final修饰成员变量

非 static 的初始化方式：

  1. 在变量声明的时候赋值
  2. 在变量所在的类的构造函数中赋值
  3. 在非静态的代码块中赋值

static 的初始化方式：

  1. 在变量声明的时候赋值
  2. 在静态的代码块中赋值
    p.s. 不能在构造方法中赋值

final修饰局部变量

需要在定义时赋值，或者在后续代码中赋值，仅一次有效

final方法

final方法不能被重写
p.s. 不能修饰构造方法

final类

final修饰的类不能被继承

    • final变量
      • final修饰成员变量
      • final修饰局部变量
    • final方法
    • final类
